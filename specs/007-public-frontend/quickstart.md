# Quickstart: Public Frontend

```bash
php artisan make:controller HomeController
php artisan make:controller PostController
php artisan make:controller CategoryController
php artisan make:controller TagController
php artisan make:controller AuthorController
php artisan make:controller ArchiveController
php artisan make:component PostCard
php artisan make:component SeoMeta
```

Register routes in `routes/web.php`.
